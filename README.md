# SAWO_Assignment

# Tech Stack
## Backend
- Express + MongoDB

## Frontend
- React

# localhost setup
## Backend
- Make sure node is installed
- npm install
- npm start

## Frontend
- cd client
- npm install
- npm start

# Functionality
- Passowordless authentication using SAWO API
- User can create, edit, delete tweets created by him
- Every user has a feed i.e tweets by other person.
